# Assignment 1 - Petri Net Genetic Algorithm

This repository contains the Python code for Assignment 1 on evolutionary process discovery using Genetic Algorithms.

## Project structure

- data/ contains the dataset.
- src/ contains the Python code.
- notebooks/ can be used for exploration and testing.
- results/figures/ contains generated plots.
- results/logs/ contains generated CSV logs.
- results/petri_nets/ contains generated Petri net visualizations.

## Required files

The dataset should be located at:

data/dataset_a1.txt

The provided support code should be located at:

src/support.py

The main solution file is:

src/assignment1_solution.py

## Conda setup

Create the environment:

conda env create -f environment.yml

Activate the environment:

conda activate assignment1-ga

Run the code:

python src/assignment1_solution.py

## Notes

The support file should not be modified. The solution code should be added in src/assignment1_solution.py.
